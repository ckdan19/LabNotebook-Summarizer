# Full Lab Digest Skill

Trigger this skill when the user asks for the **full weekly lab digest**, a **combined digest of all lab notebooks**, or a digest that covers **all sources** / **all notebooks**.

## What this skill does

Runs all five lab-notebook subagents in parallel, collects their summaries, and compiles them into a single Markdown file organized by source.

## Steps

1. **Launch all five subagents in parallel** in a single message (do NOT run them one at a time):

   - `tumbling-oysters-agent` — prompt: "Summarize all posts from the last 7 days. Return a structured Markdown summary with a ## header, the post titles/dates/URLs, and 2–3 sentence summaries. If there are no posts, say so explicitly."
   - `ariana-notebook-agent` — same prompt as above
   - `sams-notebook-agent` — same prompt as above
   - `grace-notebook-agent` — same prompt as above
   - `wordpress-agent` — same prompt as above

   Wait for **all five** to return before proceeding.

2. **Tally activity**: count how many sources returned at least one post ("had activity") and how many reported no posts ("no activity").

3. **Determine the date range**: use the 7-day window ending on today's date.
   - `week_start` = today minus 6 days (YYYY-MM-DD)
   - `week_end` = today (YYYY-MM-DD)

4. **Assemble the combined digest** using this exact structure:

```
# Full Lab Digest — [week_start] to [week_end]

> [N] of 5 sources had activity this week. [M] had none.

---

## Tumbling Oysters (Steven Roberts)

[paste the tumbling-oysters-agent summary verbatim here]

---

## Ariana Huffmyer Lab Notebook

[paste the ariana-notebook-agent summary verbatim here]

---

## Sam's Notebook (Sam White)

[paste the sams-notebook-agent summary verbatim here]

---

## Grace Crandall's Notebook

[paste the grace-notebook-agent summary verbatim here]

---

## Genefish WordPress

[paste the wordpress-agent summary verbatim here]

---

> Generated by the `full-lab-digest` skill · [week_start] to [week_end]
```

   - Paste each subagent's output verbatim under its `##` header — do **not** rewrite, paraphrase, or look for cross-source patterns.
   - If a subagent reported no activity, write a single line under that section's header: `_No posts in the last 7 days._`

5. **Write the file** to:
   `~/LabNotebook-Summarizer/digests/full-lab-digest-[week_end].md`
   (e.g. `digests/full-lab-digest-2026-07-13.md`)

6. **Return the file path** to the user in the main conversation.
